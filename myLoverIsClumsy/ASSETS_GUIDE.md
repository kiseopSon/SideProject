# 앱 아이콘 및 스플래시 이미지 가이드

## 필요한 이미지 파일

`assets` 폴더에 다음 이미지 파일들을 추가해야 합니다:

### 1. 아이콘 (icon.png)
- **크기**: 1024x1024 픽셀
- **형식**: PNG (투명 배경 가능)
- **용도**: 앱 아이콘

### 2. 적응형 아이콘 (adaptive-icon.png)
- **크기**: 1024x1024 픽셀
- **형식**: PNG
- **용도**: Android 적응형 아이콘 (foreground)

### 3. 스플래시 스크린 (splash.png)
- **크기**: 1242x2436 픽셀 (또는 비슷한 비율)
- **형식**: PNG
- **용도**: 앱 시작 화면

## 이미지 생성 방법

### 온라인 도구 사용
1. **Canva** (https://www.canva.com)
   - 무료 디자인 도구
   - 앱 아이콘 템플릿 제공
   - 1024x1024 크기로 내보내기

2. **Figma** (https://www.figma.com)
   - 전문 디자인 도구
   - 무료 플랜 제공

3. **App Icon Generator** (https://www.appicon.co)
   - 앱 아이콘 자동 생성 도구

### 간단한 아이콘 디자인 아이디어
- 하트 아이콘 (💕) + "My덤벙이" 텍스트
- 분홍색 배경 (#FF6B9D)
- 둥근 모서리

### 스플래시 스크린 디자인 아이디어
- 앱 이름 "My덤벙이" 큰 글씨
- 하트 이모지 또는 아이콘
- 분홍색 배경 (#FF6B9D)
- 중앙 정렬

## 이미지 추가 후

이미지를 `assets` 폴더에 추가한 후:

```bash
# 앱 재시작
npx expo start -c

# 또는 새로 빌드
eas build --platform android --profile preview
```

## 참고

- 이미지가 없어도 앱은 작동하지만, 기본 아이콘이 표시됩니다
- 이미지를 추가하면 더 전문적인 앱처럼 보입니다
